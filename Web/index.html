Welcome to ParaView, an open-source, multi-platform, parallel scientific visualization application. Although it can be run on a single processor, ParaView was specifically designed to run on distributed parallel computers. So far, it was succesfully tested on Windows and Linux workstations, clusters and SGI Origin 200 systems. ParaView is a turnkey application written using the Visualization Toolkit and it has framework based on a unique blend of Tcl/Tk and C++. Please go here to learn more about ParaView.

ParaView is being developed by Kitware and is funded by the US Department of Energy ASCI Views program.

The goals of the project include the following:

  • Develop an open source, multi-platform visualization application,
  • Support various parallel computation environments,
  • Develop an intuitive but flexible user interface,
  • Provide an easy method for extending the application.
