Loading Data

ParaView will read several different types of data files: VTK data sets, EnSight (v. 6 or 7), VTK image data, POP data, and STL data.  Once you have a compiled copy of ParaView, run the ParaView executable.  There are two ways to open a data file in ParaView.

  1. From the Files menu, select “Open data file”.  In the dialog box that comes up, select the file containing the data set you wish to open.

  2. Go to the VTK menu, and click on the Sources submenu.  From the list, choose the reader that is appropriate to your data set.  Depending on the reader, either a dialog box will come up automatically for you to select your data file, or there will be a button on the user interface that will bring up such a dialog box.  After you have selected the file, click the “Accept” button on the user interface to load the data file.
