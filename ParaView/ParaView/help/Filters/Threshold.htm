Threshold

Thresholding allows you to select part of your data set based on the range of the scalar values.  The button labeled “Threshold” allows you to do this.  After clicking the button, specify whether you wish to operate on point data or cell data.  The use the sliders to select the upper and lower bounds for your threshold.  If you are using point data, the checkbox labeled “All Scalars” specifies whether to include a particular cell in the threshold is only part of its points have scalars that fall within the specified range.
