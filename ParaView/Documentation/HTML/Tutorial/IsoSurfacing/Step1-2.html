Step 2

Create a test source by selecting RTAnalyticSource from VTK -> Sources menu. Click Accept. RTAnalyticSource is an artificial data source which generates three-dimensional image data (also referred to as structured points) using an analytical function of the form Maximum*Gaussian*XMag*sin(XFreq*x)*sin(YFreq*y)*cos(ZFreq*z).
