Cutting and Clipping Tutorial

This tutorial demonstrates cutting and clipping. Both of these filters are used to view part of a data set. The cutting filter extracts a subset of the data by cutting it with a plane (reduces dimension by 1) whereas the clipping filter removes the data on one side of a plane.
