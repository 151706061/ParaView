Glyph

Glyphing allows you to place an object at each point in your data set.  (Use the button labeled “Glyph”.)  You must specify a glyph source – the object to display at each point in your data set.  Choose a scale mode – how to scale the glyphs.  Choose a vector mode – how to orient the glyphs.  The orientation check box specifies whether to orient the glyphs, and the scale check box specifies whether to scale the glyphs.  The scale entry box is for specifying the scale factor for the glyphs.
