Calculator

The calculator allows you to create new data arrays from existing arrays in your data.  The button labeled “Calculator” is for doing this.  After clicking the button, you must select whether you wish to operate on point data or cell data.  You must specify an array name to hold the new data array you will create.  You then create the equation (using the mathematical buttons on the interface) to calculate a new data array.  The menu buttons labeled “scalars” and “vectors” allow you to choose existing arrays to use in this calculation.
