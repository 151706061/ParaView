Downloading

To download ParaView, you need access to its CVS repository.  Do a CVS checkout to get a copy of the source code:

cvs –d :pserver:<your CVS login>@public.kitware.com:/vtk/pvcvsroot checkout Views

If you do not have a CVS login, you can use the login anonymous, which has the password paraview.

 

You need a copy of VTK, which you can download from Kitware's web site at http://www.kitware.com/vtkhtml/vtkdata/HowToGetSoftware.html.

 

You also need a copy of Tcl/Tk, version 8.2.3, which can be downloaded from http://www.scriptics.com/software/tcltk/8.2.html.
