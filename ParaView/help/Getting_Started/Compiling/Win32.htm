Win32

VTK must be built with MPI.

 

When you run pcmaker, click the “Advanced” button in the lower right-hand corner of the window.  Click the “Use MPI” checkbox and specify where the MPI include and library directories are located.  Click “OK” in the “Advanced” window and then click “OK” in the main window to run pcmaker.  Once that is done, from a cygwin shell, cd to your vtkbin/Debug directory, and type “nmake”.

To compile, open the file Views.dsw (in your Views directory) with MSVC++.  Set ParaViewExeFiles to be the active project.  Press F7 or click the compile button.
