Install and build ParaView differently depending on the operating system. Either way, you must build and install CMake, VTK, and Tcl/Tk 8.2.3 prior to compiling and installing ParaView.

Run CMake
Once this software is installed, you are ready to build and install ParaView. Run CMake, indicating the location of the ParaView source directory, and where you wish to place any ParaView object files and and any libraries (the build directory). Normally when you run CMake, you will be presented with a GUI that allows you to specify this information plus compile options, paths, etc. as shown in the following.

Once you configure the build process, you are ready to begin compiling.

  • On Unix, cd into the build directory, and type "make".
