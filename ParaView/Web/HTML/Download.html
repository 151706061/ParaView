You can either download binaries or source code archives for the latest release or access the current development distribution through CVS.

Latest Release (0.2)

    Pre-compiled binaries:

  • PVSetup.exe: intended for Windows users : http, ftp
    Source code archives:

    We recommend using ParaViewComplete since it contains the appropriate snapshot of VTK and Tcl/Tk 8.2.3 required to build VTK. However, If you want to compile ParaView against VTK you built separately, you can download ParaView alone (make sure that you use Tcl/Tk 8.2.3, other versions will not work). If you have not already done so, you will need to download and compile/install CMake to compile ParaView. Also, on Windows, you will have to install the HTML Help library from Microsoft. See http://msdn.microsoft.com.

    CVS:

  • ParaViewComplete: contains ParaView, VTK and Tcl/Tk 8.2.3:

    cvs -d :pserver:anonymous@public.kitware.com:/cvsroot/ParaView login (respond with password paraview)

    Follow this command by checking out the source code:

    cvs -d :pserver:anonymous@public.kitware.com:/cvsroot/ParaView co -r ParaView-0-2 ParaViewComplete

  • ParaView: contains ParaView, you will have to download VTK and Tcl/Tk 8.2.3 separately:

    cvs -d :pserver:anonymous@public.kitware.com:/cvsroot/ParaView login (respond with password paraview)

    Follow this command by checking out the source code:

    cvs -d :pserver:anonymous@public.kitware.com:/cvsroot/ParaView co -r ParaView-0-2 ParaView

  • VTK: VTK snapshot for ParaView 0.2:

    cvs -d :pserver:anonymous@public.kitware.com:/cvsroot/VTK login (respond with password vtk)

    Follow this command by checking out the source code:

    cvs -d :pserver:anonymous@public.kitware.com:/cvsroot/ParaView co -r ParaView-0-2 VTK

Current development distribution

Download ParaView Complete (contains ParaView, VTK and Tcl/Tk 8.2.3) source via CVS access:

cvs -d :pserver:anonymous@public.kitware.com:/cvsroot/ParaView login
(respond with password paraview)

Follow this command by checking out the source code:
cvs -d :pserver:anonymous@public.kitware.com:/cvsroot/ParaView co ParaViewComplete

Alternatively, if you already have VTK, download ParaView source alone via CVS access (you will need Tcl/Tk 8.2.3, other versions will not work):

cvs -d :pserver:anonymous@public.kitware.com:/cvsroot/ParaView login
(respond with password paraview)

Follow this command by checking out the source code:
cvs -d :pserver:anonymous@public.kitware.com:/cvsroot/ParaView co ParaView

If you have not already done so, you will need to download and compile/install CMake to compile ParaView. Also, on Windows, you will have to install the HTML Help library from Microsoft. See http://msdn.microsoft.com.

(Note: CVS is a source code revision control system used by many participants in the open-source community. To use CVS, you must have it installed on your system. You may wish to use the Cygwin tools on Windows platforms, or WinCVS which provides a very nice GUI to CVS.
