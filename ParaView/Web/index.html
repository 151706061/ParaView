Welcome to ParaView, an open-source, multi-platform, parallel scientific visualization application. Although it can be run on a single processor, ParaView is specifically designed to run on distributed parallel computers. ParaView has been succesfully tested on Windows and Linux workstations, clusters and SGI Origin 200 systems. ParaView is a turnkey application written using the Visualization Toolkit and it has framework based on a unique blend of Tcl/Tk and C++. Please go here to learn more about ParaView.

ParaView is being developed by Kitware in conjunction with Jim Ahrens of the Advanced Computing Laboratory at Los Alamos National Laboratory. ParaView is funded by the US Department of Energy ASCI Views program.

The goals of the project include the following:

  • Develop an open-source, multi-platform visualization application.
  • Support parallel computation environments.
  • Create an open, flexible, and intuitive user interface.
  • Develop an extensible architecture based on open standards.
